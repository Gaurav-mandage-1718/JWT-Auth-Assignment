\# JWT Authentication Assignment



Full stack authentication and role-based access control project using Spring Boot backend and React TypeScript frontend.



\## Features



\- User registration

\- User login

\- JWT token based authentication

\- Role based access control

\- USER and ADMIN roles

\- Protected backend APIs

\- React dashboard based on logged-in role

\- Swagger API documentation



\## Tech Stack



\### Backend



\- Java 17

\- Spring Boot

\- Spring Security

\- JWT

\- Spring Data JPA

\- Hibernate

\- MySQL

\- Lombok

\- Swagger/OpenAPI

\- Maven



\### Frontend



\- React

\- TypeScript

\- Vite

\- React Router

\- Axios

\- React Hook Form

\## Security Notes



\- `application.properties` is ignored for security.

\- `application-example.properties` is provided as sample configuration.

\- JWT token must be sent in the Authorization header for protected APIs.

\- Passwords are stored in encrypted form using BCrypt.
